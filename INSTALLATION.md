# Installation Guide

## For Users (Recommended)

Download the pre-built application for your platform:
- **macOS**: Download the `.dmg` from [Releases](../../releases)
- **Windows**: Download the `.zip` from [Releases](../../releases)
- **Linux**: Download the `.tar.gz` from [Releases](../../releases)

No Python installation required!
